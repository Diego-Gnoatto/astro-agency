# Astro Agency Template

Template for agencies, consultancies, and professional services.

## Tech Stack
- Astro 5+ / Vue 3 / Tailwind CSS v4 / GSAP / Sitemap

## Quick Start
1. Replace all `{{PLACEHOLDER}}` values
2. `bun install && bun dev`
3. `bun run build`

## Page Structure
- `/` — Homepage (hero, services, process, pricing, FAQ, contact)
- `/sobre/` — About page
- `/orcamento/` — Quote request form
- `/orcamento/obrigado/` — Thank you page

## SEO
- Unique title, meta description, canonical URL
- Open Graph + Twitter Card
- JSON-LD Schema.org (Organization)
- Auto-generated sitemap + llms.txt
